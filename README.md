# Test Drive
Projeto de uma app cross-platform para o agendamento de test drive.

Esse pequeno projeto serve para criar uma base em programação mobile com a linguagem Xamarin e depois pretendo aprender reescrever este projeto usando o .NET Maui que está vindo pra 
substituir.

Esse projeto não visa trabalhar muito as questões de segurança de aplicações e sim mais o conceitos usados para programar em Xamarin, entender como alguns controles funcionam e algumas boas
práticas da plataforma.
